# AI Goal Planner

A web-based goal-tracking app with AI-powered coaching and planning.

## Features
- Goal tracking with status management
- 3 AI agents: Focus Coach, Strategy Agent, Progress Reviewer
- Freemium model: Free tier (3 goals), Premium (unlimited)
- User authentication with secure password hashing

## Tech Stack
- Backend: Python (custom HTTP server)
- Database: SQLite
- Frontend: HTML/CSS

## Setup
1. Clone the repo
2. `pip install flask` (if upgrading)
3. `python run.py`
4. Visit http://localhost:5001
